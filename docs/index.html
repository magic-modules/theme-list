@magic-modules/theme-list

this is the @magic-modules ThemeList component. It shows a list of all @magic-themes.

installation

npm install magic-modules/theme-list

require

first add the component to the assets

// assets/index.jsmodule.exports = {  // ..other exports  ThemeList: require('@magic-modules/theme-list'),}

usage

in a page or component

module.exports = {  View: () => ThemeList,}
